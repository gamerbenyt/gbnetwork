What is GB NETWORK?

GB NETWORK is a Minecraft server for Minecraft Java Edition. It is designed to make players feel like they have freedom to play whatever they want! 

User friendly

GB NETWORK has no pay to win features and only has crates that you earn from playtime, daily logins, and more. 

Up to date

Our servers are up to date and will run on 1.20.4, but kept available for earlier and later versions. Join now under gamerbenyt.mc-server.net

Fast support

We try our best to support our communities needs, if there is a gamemode you want to see, LET US KNOW! 

How to join

Learn how to join our Minecraft Server!

1

Add the server

Go to Multiplayer in Minecraft, then Add Server
2

Enter server address

Enter gamerbenyt.mc-server.net in the server address field and click "Done".
3

You're done!

Now you can play on the GB NETWORK with other players!
Story about GB NETWORK

The GB NETWORK is a server inspired by InvadedLands and Hypixel.  The Network is hosted AT HOME by Gamer Ben, but port forwarded to make sure it is accessible for all players.

How Come I Can't Connect?

GB NETWORK is NOT a 24/7 Minecraft server and may be down sometimes. Please contact support if you know the server is online but you are still having issues.
